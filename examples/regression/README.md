# Regressão

Lista de exemplos de regressão com breve descrição e link para cada `.Rmd`.

- [reg_dtree](reg_dtree.md): Árvore de decisão para regressão; particiona o espaço e estima por média na folha; interpretável.
- [reg_knn](reg_knn.md): k‑vizinhos mais próximos para regressão; prediz média (ou ponderada) dos k vizinhos.
- [reg_mlp](reg_mlp.md): Perceptron Multicamadas (rede neural) para regressão.
- [reg_rf](reg_rf.md): Random Forest para regressão; média de várias árvores; reduz variância.
- [reg_svm](reg_svm.md): SVR; função com margem insensível a erro até `epsilon` e penalização `cost`.
- [reg_tune](reg_tune.md): Busca de hiperparâmetros para modelos de regressão conforme `ranges`.
