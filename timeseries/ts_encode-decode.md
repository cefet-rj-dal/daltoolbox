### data sampling


```r
samp <- ts_sample(ts, test_size = 5)
train <- as.data.frame(samp$train)
test <- as.data.frame(samp$test)
```

### Model training


```r
auto <- autoenc_ed(5, 3)
auto <- fit(auto, train)
```
